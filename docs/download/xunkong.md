# 寻空

> Scighost 的原神工具箱，带给旅行者们方便与乐趣。

## 环境要求

寻空仅支持 x64 和 arm64 架构的 Windows 10 1809 及以上版本的系统，建议升级到最新版获得更稳定的体验。

## 安装

商店版和侧载版可以共存，但是因为公用一个数据库，二者版本不同时可能存在冲突。

不建议同时安装商店版和侧载版，在进行某些操作时会频繁弹窗影响体验。
